Make a new Bezier Segment.

Note
O(p^2 * r) where p is the number of path points and r is the resolution
Parameters
icorrectionThe correction for pilons segment
istopThe stop for pilons segment
path_pointsThe points for the bezier curve
resolutionThe resolution of the bezier curve, if left at 0 becomes length of path points times 3. It is important not to set the resolution too high as it can become too computationally intensive
look_ahead_distanceThe look ahead distance for the pure pursuit algorithm (do not set below 3 inches)
