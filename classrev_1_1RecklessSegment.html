virtual SegmentStatus rev::RecklessSegment::step ( OdometryState current_state)
pure virtual

Calculate the next step.

This method is executed once per step of the controller, until the segment is completed, beginning with the same cycle as init(), immediately after.

Upon reaching a return value of NEXT, the controller is intended to step to the next segment, or if no next segment is present, terminate execution.
