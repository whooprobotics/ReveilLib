{\bfseries{Doxygen Awesome}} is a custom {\bfseries{CSS theme for Doxygen HTML-\/documentation}} with lots of customization parameters.\hypertarget{md_doxygen_awesome_css_README_autotoc_md27}{}\doxysection{Motivation}\label{md_doxygen_awesome_css_README_autotoc_md27}
I really like how the Doxygen HTML-\/documentation is structured! But IMHO it looks a bit outdated.

This theme is an attempt to update the visuals of Doxygen without changing its overall layout too much.\hypertarget{md_doxygen_awesome_css_README_autotoc_md28}{}\doxysection{Features}\label{md_doxygen_awesome_css_README_autotoc_md28}

To use the theme when generating your documentation, bring the required CSS and JS files from this repository into your project.

This can be done in several ways\+:


\begin{DoxyItemize}
\item manually copying the files
\item adding the project as a Git submodule
\item adding the project as a npm/xpm dependency
\item installing the theme system wide
\end{DoxyItemize}

All theme files are located in the root of this repository and start with the prefix {\ttfamily doxygen-\/awesome-\/}. You may not need all of them. Follow the install instructions to figure out what files are required for your setup.\hypertarget{md_doxygen_awesome_css_README_autotoc_md31}{}\doxysubsection{Git submodule}\label{md_doxygen_awesome_css_README_autotoc_md31}
For projects which use git, add the repository as a submodule and check out the desired release\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{git submodule add https://github.com/jothepro/doxygen-\/awesome-\/css.git}
\DoxyCodeLine{cd doxygen-\/awesome-\/css}
\DoxyCodeLine{git checkout v2.2.1}

\end{DoxyCode}
\hypertarget{md_doxygen_awesome_css_README_autotoc_md32}{}\doxysubsection{npm/xpm dependency}\label{md_doxygen_awesome_css_README_autotoc_md32}
In the npm ecosystem, this project can be added as a development dependency to your project\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{cd your-\/project}
\DoxyCodeLine{npm install https://github.com/jothepro/doxygen-\/awesome-\/css\#v2.2.1 -\/-\/save-\/dev}
\DoxyCodeLine{}
\DoxyCodeLine{ls -\/l node\_module/@jothepro/doxygen-\/awesome-\/css}

\end{DoxyCode}


Similarly, in the \href{https://xpack.github.io}{\texttt{ x\+Pack}} ecosystem, this project can be added as a development dependency to an \href{https://xpack.github.io/xpm/}{\texttt{ {\ttfamily xpm}}} managed project.\hypertarget{md_doxygen_awesome_css_README_autotoc_md33}{}\doxysubsection{System-\/wide}\label{md_doxygen_awesome_css_README_autotoc_md33}
You can even install the theme system-\/wide by running {\ttfamily make install}. The files will be installed to {\ttfamily /usr/local/share/} by default, but you can customize the install location with {\ttfamily make PREFIX=/my/custom/path install}.\hypertarget{md_doxygen_awesome_css_README_autotoc_md34}{}\doxysubsection{Choosing a layout}\label{md_doxygen_awesome_css_README_autotoc_md34}
