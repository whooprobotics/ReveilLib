Detailed Description

template<typename MassDim, typename LengthDim, typename TimeDim, typename AngleDim>
class rev::RQuantity< MassDim, LengthDim, TimeDim, AngleDim >

Basic template class defining units.

This implementation of units is taken from OkapiLib, which in turn based it on Benjamin Jurke's work in 2015. His original blog post and code can be found at https://benjaminjurke.com/content/articles/2015/compile-time-numerical-unit-dimension-checking/
